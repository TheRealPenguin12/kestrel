# Kestrel

Kestrel is a programming language with a simple syntax, designed for beginners who just started coding text.

## Syntax

### Commenting

Commenting is a way to leave notes to yourself. The `lexer` will ignore comments.

```
& This is a comment!
& This is also a comment!
```

_Commenting is currently __under development__._
__Bug discovered by [@TheRealPenguin](https://github.com/TheRealPenguin12)__

### Output

#### Print

```
print: "Hello, World"
```

### Input

#### Input

```
input: "What's your name? "
```

### Math

#### Addition

```
print: 867485 + 756
```

#### Subtraction

Subtraction.

```
print: 7 - 5
```
#### Multiplication

```
print: 67 * 372
```

```
print: 67(372)
```

#### Division

```
print:  6 / 2
```

### Other

#### Waiting

```
wait: 2
```

#### Exiting

```
exit
```

### Comparisons

### Greater than

```
print: 6 > 4
```

### Less than

```
print: 4 < 6
```

### Equal to

```
print: 6 == 6
```

### Not equal to

```
print: 6 != 7
```
